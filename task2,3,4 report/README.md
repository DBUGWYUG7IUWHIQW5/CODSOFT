# CODSOFT Machine Learning Internship

This repository contains machine learning tasks completed as part of the
CODSOFT Machine Learning Internship.

## Task 2: Credit Card Fraud Detection
- Built a model to classify transactions as fraudulent or legitimate
- Used Logistic Regression
- Worked with pre-split datasets (fraudTrain.csv, fraudTest.csv)
- Evaluated using accuracy, precision, recall, and confusion matrix

## Task 3: Customer Churn Prediction
- Developed a churn prediction model for subscription-based services
- Performed data preprocessing and feature encoding
- Evaluated model using classification metrics

## Task 4: Spam SMS Detection
- Built an NLP-based spam detection system
- Used TF-IDF for text vectorization
- Applied Multinomial Naive Bayes classifier
- Achieved high accuracy in spam classification
